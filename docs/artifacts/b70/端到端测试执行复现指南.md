## 4. 常见失败解释

### `Node 'Int' not found`

说明 server 是旧实例，没加载到 `ComfyLiterals`。  
重启 8188 上的 ComfyUI 实例。

### `Required input is missing`

说明 workflow -> prompt 转换缺少 widget-only 节点映射。  
确认当前 `workflow_to_prompt.py` 已包含本次补丁。

### `Invalid image file: texture_fur.png` / `leather_sofa.png`

说明原始 workflow 依赖的外挂贴图素材未恢复。  
smoke 验证可使用本次脚本里的 alias；full fidelity 仍需恢复原素材。

### `UR_RESULT_ERROR_OUT_OF_RESOURCES`

说明已进入 full-size geometry 的真实 XPU 资源上限，不是再加一个 cleanup 节点就能解决的问题。
