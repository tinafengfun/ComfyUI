# B70 部署指南

## 1. 环境准备

```bash
cd /home/intel/tianfeng/comfy/ComfyUI
. .venv-xpu/bin/activate
```

确认：

```bash
python - <<'PY'
import torch
print('xpu_available', torch.xpu.is_available())
PY
```

## 2. 资产准备

```bash
bash script_examples/dasiwa_b70_prepare_assets.sh
```

该脚本会：

1. 搜索本地、remote、HF、HF mirror、ComfyICU、Civitai、ModelScope
2. 盘点 workflow 所需 custom nodes / models
3. clone/update 已知 custom node repo
4. stage 本地模型
5. 应用已知 XPU patch
6. 安装 B70 smoke 运行所需输入样例和 Qwen 模型

## 3. 启动 ComfyUI

```bash
python main.py \
  --listen 127.0.0.1 \
  --port 8188 \
  --disable-ipex-optimize \
  --lowvram \
  --reserve-vram 1.5 \
  --input-directory input \
  --output-directory docs/artifacts/b70/generated
```

## 4. 启动后检查

```bash
curl -sf http://127.0.0.1:8188/system_stats | python -m json.tool | sed -n '1,80p'
python - <<'PY'
import json, urllib.request
obj = json.load(urllib.request.urlopen('http://127.0.0.1:8188/object_info'))
for key in ['Int', 'Prompt_Edit', 'Qwen3_VQA', 'WanMultiFrameRefToVideo']:
    print(key, key in obj)
PY
```

如果补了 custom nodes 但 `object_info` 里还没有对应节点，先重启 server，不要继续复用旧实例。
